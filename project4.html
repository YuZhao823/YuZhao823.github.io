Fedran-非线性游戏设计

Immersive game design based on the open source novel Fedran, using Blender to build game scenes and 3D-model design, scripting and debugging in Unity engine.
依据开源小说Fedran进行沉浸式游戏设计，使用Blender搭建游戏场景和游戏元素设计，在Unity环境下进行脚本编写和调试。

The Context
背景介绍 1.0

非线性游戏(Non-linear Game)是指根据玩家行动产生变化的多分支叙事(Branching Storytelling)，允许玩家按照自己的想法来进行的非线性关卡(Nonlinear Level)，独立于玩家行动存在的沙盘式世界(Sandbox World)的游戏。

The Context
背景介绍 1.1

Fedran是一个共享的想象世界，为小说、故事和诗歌设定了构思背景。这个世界结合了一个以魔法使用为基础的渐渐消逝的社会，而这个社会正在受到科技和新机器的崛起的挑战。Fedran网站提供了丰富的小说内容，作为本次游戏的背景、元素、声音、空间、美学等设计灵感来源。

Initial Ideas
初步构想 1.0

我选择了Fedran中的《Sand and Blood》作为游戏设计的背景来源，在阅读小说时，我把对小说的第一印象分为了建筑，环境氛围，物体，角色，场景和其他等6个层面，依据文字描述和自我想象去构建游戏的场景空间，元素概念和玩法，作为每个场景中when, what, where, why and how的参考依据。

3D Modell
模型构建 1.0：场景搭建

我在blender中构建了地形和其他游戏对象，并尝试调整光照和渲染风格，使之变得明亮和柔和，这些颜色、色调和饱和度为观众提供了愉快和轻松的氛围，可爱的低聚物风格的对象减少了渲染工作量，使玩家能够享受流畅的游戏体验。这些色彩、色调和饱和度为观众提供了放松愉悦氛围，低多边形风格的物体减少了渲染的工作量，使玩家可以享受流畅的游戏体验。

Game Mechanic
游戏机制 1.0

小说中一个设定是家族审判的成功与否，取决于每个宗族成员的投票石总数。这帮助我建立了核心机制，在游戏中收集投票石来获得成就。我把投票石改成了水晶，因为水晶在游戏中更突出。玩家需要在地图中行走，并在某些困难的区域使用战术来获得水晶。水晶也会成为一个很好的路标，为玩家指引方向。我的目的是用这种方式来吸引玩家更积极地探索风景，与环境互动。当他们成功获得水晶后，还能获得成就感。

Game Mechanic
游戏机制 1.1

在场景中，主要有6个区域，每个区域都设计了地形和环境。在地图的不同区域，互动模式和物品收集模式也不尽相同，使游戏更加多样化和有趣。在游戏中，水晶被隐藏或设置在某些秘密和难以到达的地方。玩家需要利用地形、移动平台和破坏物品来收集水晶。
